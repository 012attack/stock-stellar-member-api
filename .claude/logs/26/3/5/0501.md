## 작업 내용

## API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/stocks/{stockId}/daily-prices` | 종목 ID로 일별 시세 조회 |
| GET | `/api/stocks/code/{stockCode}/daily-prices` | 종목코드로 일별 시세 조회 |

## 패턴 준수
- Finder 패턴: stockCode → stockId 변환 시 StockFinder.findByCode() 사용
- Command/Query 분리: Query만 구현 (데이터 입력은 외부 Python에서 처리)
