# Daily Top30 Records API - stockId 필터 추가

## 작업 내용
기존 Daily Top 30 Records 조회 API에 `stockId` 파라미터를 추가하여, 특정 종목 ID로 해당 종목의 이력만 조회할 수 있는 기능 구현

## 수정 파일
1. `adapter/webapi/record/GetDailyTop30RecordListApi.kt` - `stockId` 요청 파라미터 추가
2. `application/required/record/DailyTop30RecordLister.kt` - 인터페이스에 `stockId` 파라미터 추가
3. `application/impl/record/query/QueryDailyTop30RecordLister.kt` - `stockId`를 `hasFilters` 체크 및 repository 호출에 추가
4. `application/provided/DailyTop30RecordRepository.kt` - 모든 필터 쿼리에 `AND (:stockId IS NULL OR s.id = :stockId)` 조건 추가
5. `api_documents/record.md` - API 문서에 `stockId` 파라미터 설명 및 예시 추가

## 사용법
```
GET /member-api/api/daily-top30-records?stockId=1
GET /member-api/api/daily-top30-records?startDate=2026-02-01&endDate=2026-02-04&stockId=1
GET /member-api/api/daily-top30-records?stockId=1&page=0&size=20
```
