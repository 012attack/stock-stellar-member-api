# 첨부파일(Attachment) 관리 기능 구현

## 작업일시
2026-02-28

## 작업 내용
첨부파일 업로드/다운로드/삭제/조회 기능 전체 구현

## 생성된 파일

### Domain Layer
- `domain/attachment/Attachment.kt` - JPA 엔티티
- `domain/attachment/AttachmentTargetType.kt` - 대상 타입 enum (STOCK, NEWS 등 9가지)
- `domain/attachment/FileType.kt` - 파일 타입 분류 enum (IMAGE, DOCUMENT, VIDEO, AUDIO, ARCHIVE, OTHER)

### Application Layer - Provided
- `application/provided/AttachmentRepository.kt` - JpaRepository

### Application Layer - Required (인터페이스)
- `application/required/attachment/AttachmentUploader.kt`
- `application/required/attachment/AttachmentFinder.kt`
- `application/required/attachment/AttachmentLister.kt`
- `application/required/attachment/AttachmentDeleter.kt`
- `application/required/attachment/AttachmentDownloader.kt`

### Application Layer - Impl (구현체)
- `application/impl/attachment/command/CommandAttachmentUploader.kt`
- `application/impl/attachment/command/CommandAttachmentDeleter.kt`
- `application/impl/attachment/query/QueryAttachmentFinder.kt`
- `application/impl/attachment/query/QueryAttachmentLister.kt`
- `application/impl/attachment/query/QueryAttachmentDownloader.kt`

### Adapter Layer - Web API
- `adapter/webapi/attachment/UploadAttachmentApi.kt` - POST /api/attachments
- `adapter/webapi/attachment/GetAttachmentDetailApi.kt` - GET /api/attachments/{id}
- `adapter/webapi/attachment/GetAttachmentListApi.kt` - GET /api/attachments?targetType&targetId
- `adapter/webapi/attachment/DownloadAttachmentApi.kt` - GET /api/attachments/{id}/download
- `adapter/webapi/attachment/DeleteAttachmentApi.kt` - DELETE /api/attachments/{id}
- `adapter/webapi/attachment/dto/response/AttachmentResponse.kt`
- `adapter/webapi/attachment/dto/response/AttachmentListResponse.kt`

### Common
- `common/config/FileStorageProperties.kt` - 파일 저장 경로 설정
- `common/service/FileStorageService.kt` - 파일 I/O 서비스 (저장/로드/삭제)

### 수정된 파일
- `application.yaml` - multipart 설정 (10MB) + file-storage 설정 추가

### 문서
- `api_documents/attachment.md` - API 문서
