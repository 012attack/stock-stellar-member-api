# Company Announcement API 구현

## 작업 일시
2026-02-28

## 작업 내용
company_announcements 테이블에 대한 목록 조회 및 상세 조회 API 구현

## 생성된 파일

### Domain
- `domain/companyannouncement/AnnouncementType.kt` - PostgreSQL announcement_type enum 매핑
- `domain/companyannouncement/CompanyAnnouncement.kt` - JPA 엔티티

### Repository
- `application/provided/CompanyAnnouncementRepository.kt` - JPA Repository (필터 조회, 상세 조회)

### Application Required (인터페이스)
- `application/required/companyannouncement/CompanyAnnouncementFinder.kt` - 상세 조회
- `application/required/companyannouncement/CompanyAnnouncementLister.kt` - 목록 조회

### Application Impl (구현체)
- `application/impl/companyannouncement/query/QueryCompanyAnnouncementFinder.kt`
- `application/impl/companyannouncement/query/QueryCompanyAnnouncementLister.kt`

### Web API
- `adapter/webapi/companyannouncement/GetCompanyAnnouncementDetailApi.kt` - GET /{id}
- `adapter/webapi/companyannouncement/GetCompanyAnnouncementListApi.kt` - GET /

### DTO
- `adapter/webapi/companyannouncement/dto/response/CompanyAnnouncementResponse.kt`
- `adapter/webapi/companyannouncement/dto/response/CompanyAnnouncementListResponse.kt`

### API 문서
- `api_documents/company_announcement.md`

## API 엔드포인트
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/company-announcements` | 공시 목록 조회 (페이징, stockId/reportNm/pblntfTy 필터) |
| GET | `/api/company-announcements/{id}` | 공시 상세 조회 |
