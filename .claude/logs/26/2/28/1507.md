# Attachment API - filePath 필드 추가

## 작업 내용
첨부파일 목록/상세/업로드 API 응답에 `filePath` 필드를 추가하여 파일의 절대 경로를 반환하도록 수정

## 변경 파일
1. **AttachmentResponse.kt** - `filePath: String` 필드 추가
2. **FileStorageService.kt** - `getAbsolutePath(filePath)` 메서드 추가 (상대 경로를 절대 경로로 변환)
3. **QueryAttachmentFinder.kt** - `FileStorageService` 주입, filePath 포함하여 응답 반환
4. **QueryAttachmentLister.kt** - `FileStorageService` 주입, filePath 포함하여 응답 반환
5. **CommandAttachmentUploader.kt** - filePath 포함하여 응답 반환
6. **api_documents/attachment.md** - API 문서 업데이트

## 추가 수정 (1534)
