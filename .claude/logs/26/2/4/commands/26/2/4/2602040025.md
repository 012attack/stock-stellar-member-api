# 작업 로그: Application 실행 오류 수정

## 명령 파일
- `.claude/commands/26/2/4/0025.md`

## 작업 내용
application 실행이 안되는 이유 찾아서 수정

## 발견된 문제

### 1. JVM 버전 문제
- **증상**: `Gradle requires JVM 17 or later to run. Your build is currently configured to use JVM 8.`
- **원인**: 시스템 기본 Java가 1.8
- **해결**: `gradle.properties` 파일 생성하여 Java 21 경로 지정
```properties
org.gradle.java.home=C:\\Users\\012at\\.jdks\\azul-21.0.10
```

### 2. kotlin-reflect 의존성 누락
- **증상**: `java.lang.NoClassDefFoundError: kotlin/reflect/jvm/ReflectJvmMapping`
- **원인**: `build.gradle.kts`에 `kotlin-reflect` 의존성 누락
- **해결**: `build.gradle.kts`에 의존성 추가
```kotlin
implementation(kotlin("reflect"))
```

## 수정된 파일
1. `gradle.properties` (신규 생성)
2. `build.gradle.kts` (kotlin-reflect 추가)

## 결과
- 애플리케이션 정상 실행
- `Started MemberApiApplication in 16.104 seconds`
- Tomcat 포트 8081, context path `/member-api`
