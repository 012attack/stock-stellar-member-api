# 로그인 시 RefreshTokenInfo 역직렬화 오류 수정

## 날짜: 2026-02-04

## 문제
로그인 시도 시 아래 오류 발생:
```
SerializationException: Could not read JSON:Cannot construct instance of `yi.memberapi.domain.token.RefreshTokenInfo` (no Creators, like default constructor, exist)
```

## 원인
- `RefreshTokenInfo`는 Kotlin data class
- Jackson이 Kotlin data class를 역직렬화하려면 `jackson-module-kotlin`이 필요
- `RedisConfig`에서 `GenericJackson2JsonRedisSerializer`가 기본 ObjectMapper를 사용하고 있어 KotlinModule이 등록되지 않음

## 수정 내용

### 1. build.gradle.kts
- `jackson-module-kotlin` 의존성 추가

### 2. RedisConfig.kt
- `KotlinModule`이 등록된 `ObjectMapper`를 생성하여 `GenericJackson2JsonRedisSerializer`에 주입

## 수정된 파일
- `build.gradle.kts`
- `src/main/java/yi/memberapi/common/config/RedisConfig.kt`
