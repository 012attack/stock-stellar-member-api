# 별점 조회 시 데이터 없으면 에러 대신 null 반환

## 프롬프트
- `.claude/prompts/26/2/23/0385.md`

## 변경 내용
별점(중요도) 조회 시 데이터가 없으면 `IllegalArgumentException`을 던지던 것을 `null`을 반환하도록 수정

## 수정 파일
- `src/main/java/yi/memberapi/application/required/ImportanceGetter.kt` - 반환 타입을 `ImportanceResponse?`(nullable)로 변경
- `src/main/java/yi/memberapi/application/impl/importance/query/QueryImportanceGetter.kt` - `.orElseThrow` 대신 `.orElse(null)`로 변경, 없으면 `null` 반환
- `src/main/java/yi/memberapi/adapter/webapi/importance/GetImportanceApi.kt` - 응답 타입을 `ResponseEntity<ImportanceResponse?>`로 변경
- `api_documents/importance.md` - 별점 조회 API 문서에 `null` 반환 케이스 추가
