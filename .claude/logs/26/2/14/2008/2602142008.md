## API Endpoints
- `POST /api/daily-top30-records/{recordId}/opinions` - 상한가 레코드에 의견 작성
- `GET /api/daily-top30-records/{recordId}/opinions` - 특정 상한가 레코드의 의견 목록
- `GET /api/opinions/records` - 전체 상한가 의견 목록
- `POST /api/news/{newsId}/opinions` - 뉴스에 의견 작성
- `GET /api/news/{newsId}/opinions` - 특정 뉴스의 의견 목록
- `GET /api/opinions/news` - 전체 뉴스 의견 목록
