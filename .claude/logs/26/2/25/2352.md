# Stock-News 연결 기능 구현

## 사용된 프롬프트
- `.claude/prompts/26/2/25/2352.md`

## 작업 내용
Stock과 News를 ManyToMany로 연결하여 어떤 종목에 어떤 뉴스들이 연관되도록 하는 기능 구현

## 변경/생성 파일

### Domain
- `domain/stock/Stock.kt` - `news` ManyToMany 필드 추가 (`stock_news` 조인 테이블)

### Repository
- `application/provided/StockRepository.kt` - `findByIdWithNews()` 쿼리 추가

### Required Interfaces (Port)
- `application/required/StockFinder.kt` - `findEntityByIdWithNews()` 메서드 추가
- `application/required/StockNewsAdder.kt` - 새 파일 (뉴스 추가 인터페이스)
- `application/required/StockNewsRemover.kt` - 새 파일 (뉴스 제거 인터페이스)

### Implementation
- `application/impl/stock/query/QueryStockFinder.kt` - `findEntityByIdWithNews()` 구현
- `application/impl/stock/command/CommandStockNewsAdder.kt` - 새 파일
- `application/impl/stock/command/CommandStockNewsRemover.kt` - 새 파일

### API Endpoints
- `POST /api/stocks/{stockId}/news` - 종목에 뉴스 추가 (AddStockNewsApi)
- `DELETE /api/stocks/{stockId}/news/{newsId}` - 종목에서 뉴스 제거 (RemoveStockNewsApi)
- `GET /api/stocks/{stockId}/news` - 종목의 연관 뉴스 목록 조회 (GetStockNewsListApi)

### DTO
- `adapter/webapi/stocknews/dto/request/AddStockNewsRequest.kt` - 새 파일
